1 de octubre de 2021

Grids CSS

Mientras Flexbox es un sistema de diagramación pensado para 1 dimensión, CSS Grid permite pensar en un sistema de diagramación en 2 dimensiones.
Permite controlar tanto la posición como las dimensiones de los elementos en una grilla.

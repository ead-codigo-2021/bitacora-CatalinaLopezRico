Diseño front-end / back-end

Front-end es la parte de un sitio web que interactúa con los usuarios, por eso decimos que está del lado del cliente.
Back-end es la parte que se conecta con la base de datos y el servidor que utiliza dicho sitio web, por eso decimos que el backend corre del lado del servidor.

Diseño front-end: Lenguajes

Consta de tres lenguajes distintos:

  • HTML: Hypertext markup language
    Tiene que ver con el contenido y la estructura.
  • CSS: Cascading style sheets
    Se encarga de la presentación.
  • JS: Javascript
    Trata del comportamiento.
